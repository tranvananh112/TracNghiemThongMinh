tiếp Debug Carousel

Debug Script

Mở Console (F12) và paste đoạn code này vào Console khi đang ở trang index.html:

// Kiểm tra element
console.log('home-quiz-grid:', document.getElementById('home-quiz-grid'));
console.log('HTML bên trong:', document.getElementById('home-quiz-grid').innerHTML);

// Kiểm tra có loop-images không
console.log('Có loop-images?', document.querySelector('.loop-images'));
console.log('Có carousel-track?', document.querySelector('.carousel-track'));
console.log('Có carousel-item?', document.querySelectorAll('.carousel-item').length);

// Kiểm tra CSS
const track = document.querySelector('.carousel-track');
if (track) {
    console.log('Carousel track styles:', window.getComputedStyle(track));
}

Hoặc thử force reload carousel:

// Force reload carousel
if (window.quizManager) {
    window.quizManager.loadHomeQuizGrid();
}
